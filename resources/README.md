# Resources Folder

Place your source files here for development and testing:

- `assa.csv` - ASSA insurer data (55 columns)
- `Fedpa.xlsx` - FEDPA insurer data (28 columns)
- `regional.xlsx` - REGIONAL insurer data (28 columns)
- `nexohub.zip` - Frontend template (optional)

These files are used for:
1. Testing the import functionality
2. Understanding data structure
3. Development reference

**Note**: These files are gitignored and should not be committed to the repository.
